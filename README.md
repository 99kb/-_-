## 1. 비화재보 데이터 분류 모델
비화재보 데이터와 실제 화재 데이터를 분류하여 이후 분석에 필요한 데이터 셋 구축
#### 기존의 분류 체계 :
신고 내용 중 **경보설비, 속보 설비**  단어가 포함된 데이터에서 **테스트, 오인주의, 훈련, 점검 문의**를 제외하는 방식    

#### 문제점 :    
비화재보 데이터와 실제 화재 데이터가 제대로 구분되지 않음. 
(실제화재 중 비화재보로 분류되는 경우 多)
